# AllSafe Website

Responsive static website created from the AllSafe company profile.

## Preview

Open `index.html` directly, or run a local web server from this folder:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Publishing

Upload the complete folder to the public web directory provided by your hosting company. No database is required. The enquiry form opens the visitor's email app and prepares a message to `sales@allsafefactory.net`.

The package also includes individual product detail pages, Google Maps, a floating WhatsApp link, domain metadata for `www.allsafefactory.net`, and the downloadable company catalogue.

Before publishing, confirm that `+966 13 833 5221` is registered on WhatsApp. If AllSafe uses a separate WhatsApp mobile number, replace the number after `wa.me/` in `index.html` and `product.html`.
